# JoreMapUI

## Install

### Clone the repo via git

```bash
$ git clone https://github.com/HSLdevcom/jore-map-ui
```

### Install dependencies

```bash
$ cd jore-map-ui && yarn install
```

## Run

### Start in development mode

```bash
$ yarn start
```

Runs at http://localhost:3000/
